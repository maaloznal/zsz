Как это работает?

VLESS Monitor использует прямое подключение к вашей 3X UI панели для получения реальной статистики о VLESS конфигурациях.

Реальное количество клиентов
Статус inbound'ов
Автоматическая проверка
Мгновенные уведомления

🛠️ Установка и развертывание

🏠 Локальный запуск

# Требования: Node.js 14+
git clone https://github.com/maaloznal/zsz.git
cd zsz
npm install
npm start

Откройте http://localhost:5000 в браузере

🌐 Развертывание на VPS

# Подключитесь к VPS
ssh root@your-vps-ip

# Установите Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# Скачайте и запустите
git clone https://github.com/maaloznal/zsz.git
cd zsz
npm install
npm install -g pm2
pm2 start server.js --name vless-monitor
pm2 startup
pm2 save

🐳 Docker развертывание

# Создайте Dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production
COPY . .
EXPOSE 5000
CMD ["npm", "start"]

# Соберите и запустите
docker build -t vless-monitor .
docker run -d -p 5000:5000 --name vless-monitor vless-monitor
